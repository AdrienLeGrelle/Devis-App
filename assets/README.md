# Assets

Placez votre icône d'application ici :

- `icon.png` - Icône pour l'application (512x512 pixels recommandé)
- `icon.icns` - Icône macOS (générée automatiquement lors du build)

Pour créer un fichier .icns depuis un PNG sur Mac :
```bash
iconutil -c icns icon.iconset
```

Ou utilisez un outil en ligne pour convertir PNG en ICNS.
